# Cadastro de Pessoas - Java Spring Boot

Projeto desenvolvido em Java com Spring Boot, com foco em backend e APIs REST.

## Tecnologias
- Java 17
- Spring Boot
- Maven
- Spring Data JPA

## Funcionalidades
- Cadastro de pessoas
- Listagem de pessoas

## Endpoints
- POST /pessoas
- GET /pessoas
